# “人的痛苦，本质上是对自己无能的愤怒。“

今天看到了一个很有意思的说法，我很认同：” 内卷的产生，源于人们对成功的认可标准过于单一。“

最近事情挺多的，有时候也感觉压力挺大的，也感觉挺累的，所以就有了今天这篇文章，不为别的，仅仅只是宣泄一下而已，该完成的事情还是得完成，该承担的责任还是要承担。

标题里的这句话，是之前的指导老师的个性签名，挺写实的，同时也挺悲慨的，或许老师他也有过这样的瞬间，所以才会把这句话写下来，亦如我现在这般......

或许我天生是一个比较敏感的人，所以每当有很多事情亟待解决时，我感到的更多的是生命的无意义感，这种虚无伴随着迷惘和游离于有意义与无意义之间的挣扎，常常让我觉得很难受。我想去更广阔的平台去看一看更优秀同辈 / 前辈，这是我一直以来进取的目的，可能是我对待交友比较严苛抑或是难以敞开心扉，所以很难碰到一个兴趣相投、职业取向相似、三观难得契合的人。幸好还算幸运，如今已经遇见，那当然是闫铭畅啦💑。或许可以说，这种 soulmate 式的关系是我在这段时间里无意义里挣扎之中的唯一救赎。

也不矫情了，该做的事情是责任，想做的事情是理想，未完成的事情是挑战。人生就是体验，每一种意识形态都值得尊重，每一个积极意向都值得鼓励。也不用羡慕别人的生存状态，既然是体验，也意味着每一种体验都是独一无二，无法复制的。不用懊悔自己为什么没有做到，也没必要自负的认为自己无所不能，能力这个东西不是仅仅只有战斗力分析里 的五个维度而已。

唯一能做好的，就是每天告诉自己要的是什么，有所舍去，同时思考怎么做好而已。

ENFP 加油嗷～～
