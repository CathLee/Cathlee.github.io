# 19 级前端组寒假培训

# 前情提要

经过短暂的三周基础培训，相信你们对前端的基础知识（html+css）有了些许了解，但想要成为一名合格的前端工程师这些远远不够。先给你们介绍一下一个前端工程师的学习线路，给你们做做参考～

根据你们现阶段的情况，我们这次为期 7 天的培训主要包括：

  1. jquery 入门
  2. boostrap 入门
  3. npm 环境搭建，了解 vue 和 webpack 的环境配置
  4. 微信小程序入门

你们可以讲一下你们目前（HTML+CSS+JS）前端基础三件套的掌握程度和难点，这节课我们主要是来看看大家的学习情况并帮助大家来进行指导。
